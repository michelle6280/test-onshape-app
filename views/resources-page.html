FAQs

Answers to questions frequently asked by Onshape beginners. Click on the underlined text to view the original question and answer exchange.
How can I attach two parts?
Use the fastened mate in an Assembly studio to attach two parts. Fastened mate operates using mate connectors to fix the parts together. Learn more about the basics of mate connectors here:
How can I move the position of a sketch inside a part studio?
Use the transform tool in a part studio to shift a sketch in any direction. The transform tool operates using line, distance, or x, y, and z coordinates to change the positioning of a part.

Watch the transform tool in action here:
How can I extrude multiple parts of the same sketch, and why isn't my sketch geometry appearing after I create it?
In order to extrude multiple sections of the same part, they must be separate sketches. These sketches must be toggled to "show" instead of "hide" in order to appear as an option for extrusion.
