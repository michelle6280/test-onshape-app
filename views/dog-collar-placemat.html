Dog Collar Placemat

Use Onshape to create a dog collar!

Introduction

This dog collar was created in just a few simple steps using several part studios and a few basic tools.

You Try!

Useful tools for this project are circle, line, spline, mirror, and extrude.

Onshape Tip: Changing the color of a sketch

Right click on any part of the collar and select "edit appearance." In the color selection menu, select the color of your choice. Click the green check when you are satisfied with your selection.
